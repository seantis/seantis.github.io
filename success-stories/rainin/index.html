Pipetten weltweit online kaufen

Die Ländererkennung stellt sicher, dass alle Benutzer die passenden Pipetten finden.

Kunde:

RAININ Instrument


Technologie:

PHP

Aufgabe

Das Produktesortiment von Rainin ist je nach Land unterschiedlich. Um Besuchern online das passende Sortiment von Pipetten anzeigen zu können, müssen diese je Ländern segementiert werden.

Umsetzung

Die Ländererkennung von Benutzern stellt sicher, dass jeder Benutzer das passende Sortiment präsentiert bekommt. Die Ländererkennung geschieht anhand der IP eines Benutzers. Kann kein Land erkannt werden, so macht der Besucher selbst eine entsprechende Auswahl. Die Ländererkennung ist dem Content Management System vorgeschaltet.

Kundennutzen

Mit der Ländererkennung ist sichergestellt, dass alle Benutzer Online das passende Angebot und die passenden Preise finden.

myControl.aero Version 3.4

Version 3.4 of myControl.aero is out

See the list of changes below and enjoy logging your flights with myControl.aero. Changes and improvements:

  • Add Excel/CSV import/export for simulator flights
  • Add period to simulator flights view
  • Show subscription prices in EUR/USD
  • Allow the user to bypass the domain check on signup
  • Move aircraft statistics (hours flown) to stats view

Visit myControl.aero: https://www.myControl.aero

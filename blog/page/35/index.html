Web 2.0 - Droge, Religionsersatz oder Zukunftstechnologie?

Web 2.0: Das Internet wird von einer Ansammlung an Webseiten zur Plattform. Was das genauer heisst, ist nicht so ganz klar. Der Übergang von Web 1.0 zu Web 2.0 manifestiere sich zwischen:

  • Netscape und Google
  • Encyclopedia Britannica Online und Wikipedia
  • Publizieren und Partizipieren.

Ist damit auch noch der letzte Rest von Klarheit beseitigt? Wer nicht so genau versteht, welche revolutionären Neuerungen sich hinter dem trendigen Term verstecken, der ist wohl noch zu sehr in den Web 1.0 Denkstrukturen verhaftet. Web 2.0 eine wirkliche Innovation oder einfach alter Wein in neuen Schläuchen?

Für alle die es etwas genauer wissen wollen - ein lesenswerter Artikel der NZZ, sowie die etwas konfuse Definition von Tim O'Reilly:

http://www.nzz.ch/2006/01/13/em/articleDHFG7.html

http://www.oreillynet.com/pub/a/oreilly/tim/news/2005/09/30/what-is- web-20.html

Wetter-Alarm

Die kantonalen Gebäudeversicherungen bieten in Zusammenarbeit mit SF Meteo eine neue Dienstleistung an. Per SMS werden Warnungen vor drohenden Stürmen und Unwettern verschickt. Die Warnungen können geographisch eingegrenzt werden und sollen so eine hohe Zuverlässigkeit bieten.
Die Versicherer versprechen sich von diesem Service kleinere Schadensummen.

http://www.wetteralarm.ch

Wie cool ist "cuil"?

Einmal mehr wird im Internet eine neue Google-Konkurrenz angekündigt. Die neue Suchseite heisst "cuil" (sprich "cool").

Laut eigenen Angaben verfügt "cuil" über den, mit rund 120 Milliarden erfassten Seiten, deutlich grössten Suchindex aller bekannten Suchmaschinen. Google habe nur 40 Milliarden.

Auch würden von "cuil" keine Nutzerdaten gesammelt. Dies wurde Google in jüngster Vergangenheit wiederholt vorgeworfen.

http://www.cuil.com

wikipedia - oder am Anfang war eine gute Idee

Die freie Enzyklopädie Wikipedia scheint in  einer Sinnkrise - vielleicht gar einer Mid-Life Crisis - zu stecken. So machten fragwürdige Artikel zum Thema Hitler oder DDR in der deutschen Version Schlagzeilen. Im englischen "wiki" tauchten verschwörerische Artikel zum Kennedy-Mord auf.
Zur Disskusion steht nun offenbar eine Registrierungspflicht. Ob eine solche Registrierung jedoch noch der Grundidee einer freien Enzyklopädie entspricht, ist mehr als fraglich.

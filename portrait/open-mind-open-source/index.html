Open Mind - Open Source

Mit Open Source verbinden wir mehr als einfach "Gratis-Software". Der Ansatz entspringt vielmehr einer Geisteshaltung und damit verbundenen Idealen auf dem Weg zu besseren IT-Lösungen.

Der Begriff Open Source wird unterschiedlich verwendet. Wir verstehen darunter in erster Linie den "Open Mind" in jeder Beziehung und halten uns an die Open Source Definition.

Wir hosten folgende und viele andere Open Source Projekte auf GitHub: 

  • https://github.com/seantis
  • https://github.com/onegov

Eine Auswahl aus den eingesetzten Open Source Technologien:

  • Python ist eine objektorientierte Scriptsprache, damit umgesetzt sind unter anderem auch Zope bzw. Plone.
    www.python.org
  • Pyramid ist ein leichtgewichtiges Web-Applikations-Framework.
    docs.pylonsproject.org
  • Zope (Z Object Publishing Environment) ist ein objektorientierter, in der Programmiersprache Python geschriebener, quelloffener Webanwendungs-Server.
    www.zope.org
  • Plone ist ein Enterprise Content Management System, geschrieben in der Programmiersprache Python. Es kann für Intranet- und Extranet-Anwendungen, als Document Publishing System und als Groupware-Tool eingesetzt werden.
    www.plone.org
  • Puppet ist ein Systemkonfigurationswerkzeug zum Konfigurationsmanagement. 
    http://puppetlabs.com
  • Git ist eine Freie Software zur verteilten Versionsverwaltung
    git-scm.com
  • Ubuntu setzen wir als Betriebssystem (OS) auf unseren Servern ein. Ubuntu verwendet den Linux-Betriebssystemkern, die meisten grundlegenden Systemwerkzeuge stammen vom GNU-Projekt. 
    www.ubuntu.com
  • Apache ist der am meisten verbreitete Webserver im Internet. Der Modulare Aufbau ermöglicht sichere verschlüsselte Verbindungen zwischen Server und Browser (SSL), sowie den Betrieb als Proxy-Server.
    www.apache.org
  • MySQL ist ein multithreaded, multi-user, SQL (Structured Query Language) Database Management System. MySQL ist die im Webbereich am meisten verwendete Datenbank.
    www.mysql.com
  • PostgreSQL ist ein freies, objektrelationales Datenbankmanagementsystem (ORDBMS).
    www.postgresql.org
  • Nginx ist ein high-performance HTTP server und reverse proxy. Nginx ist bekannt für seine hohe Performance, Stabilität, Vielfalt an Funktionen und Konfigurationsmöglichkeiten.
    wiki.nginx.org
  • Trac ist ein webbasiertes Projektmanagement-Werkzeug zur Softwareentwicklung.
    trac.edgewall.org
  • Apache Subversion (SVN) ist eine Freie Software zur Versionsverwaltung von Dateien und Verzeichnissen. Die Versionierung erfolgt in einem zentralen Projektarchiv (engl. repository) in Form einer einfachen Revisionszählung.
    subversion.apache.org
